# simple-react-webpack
A nice webpack and redux / react quick start setup


## Install
`npm i`

## Run
`npm start`

## Notable Contents
* babel for ES6 support
* immutablejs
* mocha/chai
* postcss
* react
* react-dom
* react-redux
* react-router
* react-templates
* redux
* redux-devtools
